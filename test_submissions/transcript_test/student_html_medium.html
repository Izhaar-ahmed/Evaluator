Supervised learning uses labeled datasets where each input is paired with a known output. The model learns this relationship to predict outcomes for new data. It is used when labeled data is available.

A common example is the Iris dataset for classification.

Key libraries include NumPy, Pandas, Matplotlib, Seaborn, and Scikit-learn.

The workflow involves loading data, converting to a DataFrame, visualizing (e.g., t-SNE), and preprocessing (cleaning and standardization).

Data is split into training and testing sets, followed by model training using algorithms like Logistic Regression.

Performance is evaluated using accuracy score, classification report, and confusion matrix.
