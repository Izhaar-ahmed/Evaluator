In today's class on supervised learning, we studied the Iris dataset using scikit-learn and Google Colab. The key libraries imported were NumPy for numerical operations, Pandas for data manipulation and DataFrames, Matplotlib for basic plotting, and Seaborn for advanced visualizations like heatmaps. We loaded the Iris dataset which has 3 classes (Setosa, Versicolor, Virginica) with 4 features: sepal length, sepal width, petal length, and petal width. We used t-SNE for dimensionality reduction to visualize the data in 2D. StandardScaler was used for data preprocessing. The logistic regression model was trained with 80-20 train-test split and achieved 100% accuracy. We also learned about precision, recall, F1 score and the confusion matrix for model evaluation.
